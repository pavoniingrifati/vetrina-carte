Admin giornata

Incolla il JSON/CSV estratto dagli screen di Football Manager, aggiorna le statistiche dei giocatori e scarica il database aggiornato.

1. Database giocatori

Puoi usare il database base oppure caricare un players_stats.json delle giornate precedenti.

2. Incolla dati giornata

Accetta JSON o CSV. Colonne supportate: name/giocatore, vote/voto, goals/gol, assists/assist, ownGoals/autogol, yellow/giallo, red/rosso, played/ha_giocato, points/punteggio.
