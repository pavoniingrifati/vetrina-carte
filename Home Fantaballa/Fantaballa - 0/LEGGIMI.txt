FANTABALLA WORLD CUP - VERSIONE CLASSIFICA CON BOTTONE

Apri index.html per usare il gioco.

Cosa e' stato modificato:
- In home ora c'e' solo il bottone "Classifica".
- Cliccando il bottone si apre classifica.html con la classifica completa.
- La classifica viene caricata dal file data/classifica.json.
- Il bottone vittoria via email resta attivo solo in caso di vittoria finale.
- Email vittoria: pavoniingrifati@gmail.com

Come aggiornare manualmente la classifica:
1. Apri data/classifica.json.
2. Modifica o aggiungi righe dentro la lista "classifica".
3. Ogni riga deve avere:
   - "nome_squadra"
   - "campione_del_mondo"
4. Salva il file.
5. Apri classifica.html oppure clicca "Classifica" dalla home e premi "Aggiorna".

Esempio:
{
  "aggiornato_il": "2026-07-08",
  "classifica": [
    { "nome_squadra": "Pavoni Ingrifati", "campione_del_mondo": 3 },
    { "nome_squadra": "Fantaballa XI", "campione_del_mondo": 1 }
  ]
}

Nota:
Se apri index.html direttamente dal computer, alcuni browser possono bloccare il caricamento del file JSON.
Quando carichi la cartella su un sito/hosting, il caricamento funziona normalmente.
